`id`
:   (Required, string) The ID of the document that contains the vector to use as the query vector.

`index`
:   (Required, string) The name of the index containing the source document.

`path`
:   (Required, string) The field path of the `dense_vector` field in the source document.

`routing`
:   (Optional, string) A custom routing value to use when retrieving the source document.
